# PT HORA HORE Landing Page

Landing page modern untuk PT HORA HORE - Solusi Cloud Terdepan di Indonesia.

## Fitur
- Responsive design
- Animasi modern
- Form kontak interaktif
- Optimasi performa tinggi
- Docker containerization

## Teknologi
- HTML5, CSS3, JavaScript
- Font Awesome 6.4.0
- Animate.css 4.1.1
- Nginx Web Server
- Docker & Docker Compose

## Persyaratan Sistem
- Docker Engine 20.10+
- Docker Compose 2.0+
- 512MB RAM minimum
- 1GB storage kosong
